# Aggredate Remove Branding License

Price: **$4 per project**.

This is the tiny paid option for people who want to remove the "Powered by
Aggredate" watermark from one public project.

## What buyers get

- Permission to use `data-branding="false"` on one website, README, or product.
- Continued use of the free widget and README badge endpoint.
- No account, no dashboard, no subscription.

## What stays free

- Using Aggredate with the watermark.
- Forking the MIT-licensed code.
- Local/private/internal experiments.

## How to use it

After buying, set:

```html
<div data-aggredate
     data-github="owner/repo"
     data-branding="false"></div>
```

## Notes

This is an honesty-license, not DRM. Keep it simple until there is enough demand
to justify accounts or license keys.
